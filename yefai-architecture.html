Yefai Predictive Maintenance Architecture

MATWI görüntü/sensör verisi → FastAPI AI backend → Supabase pgvector + local disk → RAG, NovaVision, PUQ AI ve Next.js/Tauri dashboard akışı.

Backend Core

  • • FastAPI exposes anomalib, NovaVision, embeddings, predictions, chat, notifications, spare-parts and dashboard routers.
  • • Auth/org middleware protects B2B SaaS routes with organization context and RBAC.
  • • Mock/live gates are explicit: no live NovaVision, PUQ AI or LLM success is claimed without credentials/artifacts.

Data & AI

  • • MATWI image and sensor files stay on local disk; Supabase stores metadata and pgvector embeddings.
  • • Anomalib PatchCore handles image anomaly scoring; Jina CLIP v2 provides shared image/text embeddings.
  • • RAG uses vector similarity plus SQL metadata filters before sending context to Gemini or Claude.

Operations Loop

  • • Anomaly score feeds wear prediction, hours-to-critical and spare-parts crisis scoring.
  • • Crisis workflow can prepare mock PO and alternative supplier recommendations.
  • • PUQ AI receives webhook payloads for Telegram, email and SMS notifications with retry/logging.
